¿Qué es HTML5?


HyperText Markup Language 5





Se trata de la quinta revisión de HTML,

permite definir nuevos estándares de

desarrollo web, modificando el código

existente para solucionar problemas y

actualizándolo a las nuevas necesidades

de hoy en día.

HTML es un lenguaje de marcado, es

decir, un lenguaje de etiquetas que

permite incluir o hacer referencia a todo

tipo de información.







Novedades



  • Nuevos elementos multimedia como <audio> y <video>.
  • Cambio, redefinición o estandarización de elementos como <a>, <cite> o <menu>.
  • Nuevas etiquetas semánticas para estructurar los documentos HTML, destinadas a reemplazar la necesidad de tener una etiqueta <div> que identifique cada bloque de la página.
  • Integración de gráficos vectoriales (SVG) en sustitución de los genéricos <object> y un nuevo elemento <canvas> que permite dibujar en él.
  • Almacenamiento local del lado del cliente.



  • Incorporar elementos multimedia sin utilizar otros plugins, como Adobe Flash.
  • Opciones de geolocalización para mejorar la experiencia de usuario ofreciendo cambiar el idioma o redireccionar a una página específica dependiendo de la región.
  • Crear animaciones en 2D e incluir acciones específicas para el teclado, el mouse o el control de videojuegos para interactuar con una página.
  • Desarrollar aplicaciones web con diseño adaptativo que son accesibles desde cualquier tipo de dispositivos: tabletas, teléfonos inteligentes, computadoras de escritorio e incluso televisiones inteligentes.

Estructura de etiquetas

<etiqueta atributo = "valor"> contenido </ etiqueta>






Etiqueta

La parte esencial de una etiqueta HTML es la etiqueta de apertura. Consiste en

escribir el nombre de la etiqueta, colocándola entre los caracteres < y >, siempre en

minúsculas.

La mayoría de etiquetas requieren que se especifique un cierre de etiqueta para

saber dónde termina de actuar. Se escribe igual que la etiqueta de apertura, pero

con la barra / inmediatamente después del <.



Atributo

Existen atributos específicos en ciertas etiquetas HTML. Estos determinan cierta

información sobre la etiqueta y van asociados a un valor determinado.

Este par atributo-valor se escribe después del nombre de la etiqueta, separándola

por espacio y antes del carácter > de la etiqueta de apertura.



Contenido

En el interior de la etiqueta HTML debe colocarse la información

que se quiera que sea afectada por dicha etiqueta.

Una etiqueta puede tener varios pares atributo-valor, pero nunca

se debe repetir el mismo atributo en una misma etiqueta varias

veces, ya que sobreescribiría el anterior. El orden de los atributos

no importa.



Comentario

Consiste en introducir breves fragmentos de texto que el

navegador ignora y no tendrá en cuenta a la hora de crear la

página visualmente, pero que sirven para documentar detalles,

explicar algo importante o introducir algún texto relevante.

Para introducir comentarios en el código HTML, basta con escribir

los fragmentos de texto <!-- y --> entre el comentario en cuestión

que se quiera incluir.
